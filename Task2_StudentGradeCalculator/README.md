# Task 2 - Student Grade Calculator

## Overview
This is a console-based Java application developed as part of the CodSoft Java Development Internship. 
The program takes marks obtained (out of 100) in multiple subjects, calculates the total marks and 
average percentage, and assigns a grade based on the average achieved.

## Objective
Build a Student Grade Calculator that:
- Accepts marks for a given number of subjects
- Calculates total marks and average percentage
- Assigns a grade based on the average
- Displays all results to the user

## Features
- Accepts input for any number of subjects
- Calculates total marks obtained across all subjects
- Calculates average percentage
- Assigns a letter grade (A, B, C, D, E, F) based on performance
- Displays a summary of results: total marks, average percentage, and grade

## Grading Criteria
- 90 and above: A
- 80 to 89: B
- 70 to 79: C
- 60 to 69: D
- 40 to 59: E
- Below 40: F

## Tech Stack
Language: Java
Concepts Used: Loops, Conditionals, Arrays, Scanner class for user input

## How to Run
1. Clone or download this repository.
2. Navigate to the Task2_StudentGradeCalculator folder.
3. Compile the program:javac GradeCalculator.java
4. Run the program: java GradeCalculator
